# Getting Started with Oracle Knots

Welcome to Oracle Knots, your sovereign Bitcoin node with a modern interface.

## What is Oracle Knots?

Oracle Knots is a Bitcoin node client based on Bitcoin Knots, designed to make running a full node accessible and enjoyable. It combines powerful node functionality with a user-friendly graphical interface.

## Your First Steps

### 1. Start the Node
On the **Dashboard**, click the "Start Node" button. Your node will begin synchronizing with the Bitcoin network. This may take several hours depending on your connection speed.

### 2. Monitor Sync Progress
Watch the "Blockchain Sync" card on the dashboard to see progress. When sync reaches 100%, you're fully synchronized.

### 3. Create a Wallet
Go to **Wallet Manager** and click "Create New Wallet". This creates a new wallet to hold your Bitcoin.

### 4. Generate a Receive Address
In the **Wallet Manager**, go to the "Receive" tab and click "Generate Receive Address". You can now receive Bitcoin.

### 5. Configure Your Settings
Visit **Configuration** to customize your node:
- Storage settings (how much disk space to use)
- Network settings (Tor, I2P, connection limits)
- Policy (what transactions to relay)

## Key Features

- **Private**: Your data stays on your machine
- **Sovereign**: Run your own Bitcoin network rules
- **Accessible**: Modern interface designed for everyone
- **Powerful**: Full node with advanced features
